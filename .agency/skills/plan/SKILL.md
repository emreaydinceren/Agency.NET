---
description: Decompose an objective into an ordered, numbered step list before acting.
when_to_use: When you need to plan work before executing it, or when a task is complex enough that a written plan will keep the work on track.
---
Before acting on any objective, produce a numbered plan:

1. Read the objective carefully and identify every distinct deliverable.
2. Break the objective into an ordered list of concrete, individually-completable steps.
   - Each step should be small enough to verify in isolation.
   - If a step depends on a previous step, say so explicitly.
3. Write the numbered plan in the conversation **before** doing any work — this makes it visible to any downstream reviewer or Goalkeeper.
4. Execute the steps in order, checking off each one as it is completed.
5. If you discover mid-execution that the plan is wrong, update the numbered list in the conversation before continuing.

**Why keep the plan in the transcript?** The plan serves as a shared contract between you and any automated done-check (Goalkeeper). When the Goalkeeper reads the conversation, it needs to see both the plan and the evidence that each step was completed.
