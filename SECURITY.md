# Security Policy

## Supported Versions

Agency is currently pre-1.0 (`0.x`). Only the latest published release
receives security fixes; there is no long-term support branch yet.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

This table will be revised once the project reaches a stable `1.0` release.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you believe you have found a security vulnerability in Agency, report it
privately using one of the following channels:

- **Preferred:** [GitHub Security Advisories](https://github.com/emreaydinceren/Agency.NET/security/advisories/new) —
  use the "Report a vulnerability" button on the repository's Security tab.
- **Email:** 13038027+emreaydinceren@users.noreply.github.com

Please include as much of the following as you can to help us triage the
report quickly:

- A description of the vulnerability and its potential impact.
- Steps to reproduce, or a proof-of-concept.
- The affected version(s) or commit(s).

We will acknowledge receipt of your report as soon as possible and aim to keep
you informed of progress toward a fix and public disclosure.
