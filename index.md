# Agency

**Build AI agents in C# that remember, finish the job, and stay observable — not amnesiac chatbots that stop the moment they *feel* done.**

Agency is a layered .NET 10 toolkit for RAG pipelines and autonomous agents: `AGENT = LLM + HARNESS`. The LLM does the thinking; the harness — prompting, tools, memory, permissions, and the *"is it actually done?"* check — is everything else.

- 📖 **[Documentation portal](docs/Home.md)** — narrative deep-dives on the harness, memory, permissions, and ingestion, plus a per-project reference.
- 🔧 **[API Reference](api/index.md)** — generated from the XML doc comments on every public type.
- 🚀 **[60-second quick start](https://github.com/emreaydinceren/Agency.NET#-try-it-in-60-seconds)** — clone, run one script, and you're chatting with a tool-using agent.

Source, issues, and releases live on [GitHub](https://github.com/emreaydinceren/Agency.NET).
